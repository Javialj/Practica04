Etiquetas

Las etiquetas de encierre pueden hacer de una palabra o una imagen un hipervínculo a otro sitio, se pueden cambiar palabras a cursiva, agrandar o achicar la letra, etc.

-Partes principales de una etiquetas:

  • -La etiqueta de apertura: consiste en el nombre del elemento, encerrado por paréntesis angulares (< >) de apertura y cierre. Establece dónde comienza o empieza a tener efecto el elemento.
  • -La etiqueta de cierre: es igual que la etiqueta de apertura, excepto que incluye una barra de cierre (/) antes del nombre de la etiqueta. Establece dónde termina el elemento.
  • -El contenido: este es el contenido del elemento, que en este caso es sólo texto.
  • -El elemento: la etiqueta de apertura, más la etiqueta de cierre, más el contenido equivale al elemento.

-Atributos:

Los atributos contienen información adicional acerca del elemento, la cual no quieres que aparezca en el contenido real del elemento. Aparte los atributos siempre deben de contar con

  • Un espacio entre este y el nombre del elemento (o del atributo previo, si el elemento ya posee uno o más atributos).
  • El nombre del atributo, seguido por un signo de igual (=).
  • Comillas de apertura y de cierre, encerrando el valor del atributo.

Un ejemplo de este caso es el uso del atributo class.

< h2 class="Subtitulo" >

-Elementos anidados:

Puedes también colocar elementos dentro de otros elementos. Si, por ejemplo, quieres resaltar una palabra del texto, podemos encerrarla en un elemento < strong >, que significa que dicha palabra se debe enfatizar.

Un ejemplo de este caso es:

"Una palabra clave realmente importante para mi"

Nota: Debes asegurarte que los elementos estén correctamente anidados, los elementos deben abrirse y cerrarse ordenadamente, de forma tal que se encuentren claramente dentro o fuera el uno del otro.

-Elementos vacíos:

Algunos elementos no poseen contenido, y son llamados elementos vacíos, como por ejemplo el elemento < img >.

-Elemento div y las etiquetas semánticas:

El elemento < div > funciona principalmente como una especie de bloque que dentro puede tener más < div >.

Para quitar confusiones debido a tener desasidos < div >, están las etiquetas semánticas, que sirven para tener una idea general de una estructura sin tener que recurrir a lo antes mencionado, estos elementos tiene la siguiente estructura:

Javier Alejandro

Numero:#########

Correo:example@gmail.com

Imágenes

Como mencionamos anteriormente en los elementos vacíos las imágenes en un archivo html son colocadas como un elemento vacío < img >

En este elemento, parecido al elemento < a >, se usa una etiqueta para poder mostrar la imagen, que en este caso seria src, para poder mostrar la imagen los mas recomendado es el uso de una imagen descargada.

Nota: Aparte de la etiqueta src, también esta la etiqueta alt que funciona para mostrar un texto en caso de que al imagen no se muestre.

-Imagen como enlace:

Como se explico en elementos anidados dentro del elemento < a > se puede introducir el elemento < img > para que este funcione como link hacia una pagina.

Nota: Funcionamiento del elemento < a >, en enlaces.

-Ubicaciones:

Dentro del elemento < p >, para esto en el archivo css se utiliza float.

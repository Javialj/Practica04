Enlaces

Para implementar un vínculo, necesitas usar un vínculo simple < a >. Para convertir algún texto dentro de un párrafo en un vínculo

Después de introducir la etiqueta, dentro de ella le ponemos el atributo href, este puede ser un vinculo ya sea en la misma pagina(Eso se logra poniendo el atributo id a la parte de donde desea ir) o poniendo un link a alguna otra pagina

Y por ultimo dentro de la etiqueta < a > ponemos el texto que hace referencia a la pagina que vinculamos.

Ejemplo:

< a href="Aqui va la ubicacion del link" > Aqui va el texto para el enlaces < /a >

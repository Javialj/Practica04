Tablas:

Las tablas sirven para ordenar una serie de datos.

Una tabla en html se crea mediante el elemento < table >, dentro de este elemento van otros elementos:

< tr >: este elemento anidado en la tabla sirve para crear las filas.

< th >: este elemento crea columnas con el formato como si fuera un encabezado.

< td >: este elemento al igual que el < th > crea columnas sirviendo para poder rellenar datos.

Nota: No es necesario la presencia de los elementos < td > y < th > simultáneamente dentro de la tabla.
